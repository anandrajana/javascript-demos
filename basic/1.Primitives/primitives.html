Primitive Data types & Variables (Try in Console of Debugger)

Primitive Data Types

number, string, boolean, undefined , null

     
 
1. Number—this includes floating point numbers as well as integers, for example 1, 100, 3.14.
             var n = 1;
             typeof n;  //number
             var n2 = 1.23;
             typeof n2; //number

            Infinity
            It represents a number too big for JavaScript to handle.
                Dividing by 0 will give you infinity.
                 var a = 6 / 0;
                  a   // Infinity

                    NaN
                    "Not A Number", NaN is a special value that is also a number.

                    typeof NaN     //"number"

2. String—any number of characters, for example "a", "one", "one 2 three".
                 var s = "some characters";
                typeof s;  // string

3. Boolean—can be either true or false.

                   var b = true; typeof b;  //"boolean"

4. Undefined—when you try to access a variable that doesn't exist, 
                     you get the special value undefined

                     foo   //   foo is not defined

                    typeof foo  //  "undefined"

5. Null— It means no value, an empty value, nothing (null is object)

                The null value, is not assigned by JavaScript behind the scenes;
                it can only be assigned by your code.

                var somevar = null
              
                somevar // null

                 typeof somevar    //  "object"

       -  Primitive (the five types listed above), or
       -   Non-primitive (objects)

RECAP on Primitives


There are five primitive data types in JavaScript:
number, string, boolean, undefined , null
Everything that is not a primitive is an object
The number data type can store positive and negative integers or floats, hexadecimal numbers, octal numbers, exponents, and the special numbers NaN, Infinity, and –Infinity
The string data type contains characters in quotes
The only values of the boolean data type are true and false
The only value of the null data type is the value null
The only value of the undefined data type is the value undefined

Data structure based object ( Array - List of values (elements) of any type)


        To declare a variable that contains an empty array
             var a = [];
             typeof a;    //"object"

           To define an array that has three elements :
            var a = [1,2,3];
            a        // [1,2,3]
            a[0]  // 1     

            a[2] = 'three';  // update array    //[1,2,"three"]

            a[3] = 'four';    // add element
    
            var a = [1,2,3];
            a[6] = 'new';
             a ;   //  [1, 2, 3, undefined, undefined, undefined, "new"]
        
            // delete 
             var a = [1, 2, 3];
             delete a[1]; 
             a;     //    [1, undefined, 3]                

             Arrays of arrays
             ==========   
             var a = [1, "two", false, null, undefined];
             a  //   [1, "two", false, null, undefined]
            a[5] = [1,2,3]
            a    //[1, "two", false, null, undefined, [1, 2, 3]]


            2 Dimensional array:

            var a = [[1,2,3],[4,5,6]];
            a ; //   [[1, 2, 3], [4, 5, 6]]
            a[0][0]

Summary on Array

An array is a data store
An array contains indexed elements
Indexes start from zero and increment by one for each element in the array
To access array elements we use the index in square brackets
An array can contain any type of data, including other arrays

Summary on Operators


Arithmetic operators: +, -, *, /, and %.
Increment operators: ++ and --.
Assignment operators: =, +=, -=, *=, /=, and %=.
Special operators: typeof and delete.
Logical operators: &&, ||, and !.
Comparison operators: ==, ===, !=, !==, <,>, >=, and <=.
